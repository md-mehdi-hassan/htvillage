# High-Tech Village Dhaka

Premium tech/real-estate landing page for HTV Dhaka — Vite + React, Tailwind CSS, GSAP ScrollTrigger, and Lenis smooth scrolling.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Replace images

Drop your assets into `public/assets/images/` using these filenames:

- `hero-bg.jpg`
- `campus-masterplan.jpg`
- `rnd-collaboration.jpg`
- `vertical-gardens.jpg`
- `dhaka-skyline.jpg`
- `advanced-manufacturing.jpg`
- `ai-innovation-suite.jpg`
- `smart-offices.jpg`
- `innovation-campus.jpg`
- `executive-suites.jpg`
- `command-center.jpg`

All copy lives in `src/data/content.js`.
